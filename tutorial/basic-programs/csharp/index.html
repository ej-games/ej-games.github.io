C# Basic Programs

This is a big fat list of all of the basic programs to make with C# to learn it as a language. Use the list below as a sort of table of contents, ranked from easiest to hardest. When I say hardest, I don't mean it's like "oh my god this is impossible" hard. It's all doable, silly!
A sort of table of contents
  1. Hello World program
  2. Calculator
  3. Temperature Converter
  4. Number Guessing Game
  5. Fibonacci and Factorial Calculator
  6. To Do List
  7. Database Manager
  8. Weather Display
I do recommend you follow these in order if you've never programmed in this language before. Especially the first one, which should show you how to set up a working environment.
