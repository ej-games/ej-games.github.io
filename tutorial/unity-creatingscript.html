Creating a script

This is a Unity tutorial on the basics of creating a script. It will not be relevant to Vanilla C# programmers.

Intro

In order for things to happen in your game, you will probably need to encounter some C# code,
unless you want to download some Visual Scripting plugin for Unity, or use the built-in one (haven't tried it, can't form an opinion).
Don't shoot the messenger! Where the messenger is me!


Anyway, when you are programming your game in Unity, you will need to use C#. It used to be a mix of JavaScript and C#, but those days are over now. I am here to help you learn the C# that you need.

And the first place to start is the intro to C#! So go to this lovely website (just everyone's favourite, ejgames.co.uk) and follow that tutorial.

Preparation

Hopefully by now you should have learnt the basics of C# (whether it's on my website or another) and also you should have a blank Unity project.
Unless you don't have Unity! Then install that and create a new 2D project, which is what I am using.

Then, add a GameObject into your scene. Either right click in the hierarchy tab and select "2D Object → Sprites → Square" or select the "GameObject" option at the top of the screen and follow the same steps.

You should see something like this in the centre of your screen:
(Don't worry if it's not exactly the same. For example, if you just see a square on a blueish background, switch to the "Scene" tab above it)
That is pretty much all you will need to prepare for making a script. Any GameObject will do.
